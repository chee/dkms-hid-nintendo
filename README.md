# dkms-hid-nintendo

A Nintendo HID kernel module.


## Installation

Install it from source with:

```
git clone https://github.com/nicman23/dkms-hid-nintendo
cd dkms-hid-nintendo

sudo dkms add .
sudo dkms build nintendo -v 2.0
sudo dkms install nintendo -v 2.0
```
